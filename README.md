# DataScience-Phishing-URL-
Klasifikasi link Phishing Menggunakan 10 model Machine Learning
Analisis ini bertujuan untuk melihat dari 10 model yang digunakan cocok pada dataset phishing ini yang di dapat di UCI Machine Learning, penelitian sebelumnya mendapatkan nilai 99,7% . 

Link Dataset:
https://archive.ics.uci.edu/dataset/967/phiusiil+phishing+url+dataset

algoritma yang di gunakan adalah : Linear SVC, MLP Classifier, XGBoost, Logistic Regression, Random Forest, LightGBM, SGD Classifier, Decision Tree, Gaussian Naive Bayes, dan K-Nearest Neighbors.

hasil yang di dapat dari 10 model yang digunakan terdapat 3 model yang paling unggul ya itu Logistic Regression, K-Nearest Neighbors, dan MLP Classifier. Namun dari 3 model yang paling unggul MLP Classifier yang jadi pemenangnya karena di lihat di Learning Curvenya.
